# YouTube Downloader - Node.js Dependencies
# This file documents all npm packages required to run this project
# Install dependencies using: npm install

## Core Dependencies

# FFmpeg static binary - provides ffmpeg for media conversion
ffmpeg-static@^4.4.1

# Fluent FFmpeg - Node.js wrapper for FFmpeg
fluent-ffmpeg@^2.1.3

# YT-DLP wrapper - YouTube video information extraction and streaming
yt-dlp-wrap@^2.3.10

# DistTube YT-DL Core - Enhanced YouTube stream extraction
@distube/ytdl-core@^4.13.0

## Development Dependencies

# TypeScript - Language compiler
typescript@^4.5.4

# ts-node - TypeScript execution for Node.js
ts-node@^10.4.0

# TypeScript definitions for Fluent-FFmpeg
@types/fluent-ffmpeg@^2.1.27

# Node.js built-in types are included in modern Node.js versions (v18+)

## Installation Instructions

### Option 1: Using npm (Recommended for Node.js)
```bash
npm install
```

### Option 2: Manual installation
If you prefer to install packages individually:
```bash
npm install ffmpeg-static@^4.4.1
npm install fluent-ffmpeg@^2.1.3
npm install yt-dlp-wrap@^2.3.10
npm install @distube/ytdl-core@^4.13.0
npm install --save-dev typescript@^4.5.4
npm install --save-dev ts-node@^10.4.0
npm install --save-dev @types/fluent-ffmpeg@^2.1.27
```

### Option 3: Using yarn
```bash
yarn install
```

## System Requirements

- **Node.js**: v18.0.0 or higher
- **npm**: v8.0.0 or higher (or compatible package manager like yarn/pnpm)
- **ffmpeg**: Included via ffmpeg-static (optional: system ffmpeg for custom builds)

## Notes

1. This project uses TypeScript, so no Python is required
2. FFmpeg is bundled with ffmpeg-static and does not need separate installation
3. For Windows users with older systems, you may need to install ffmpeg separately if ffmpeg-static doesn't work
4. See README.md for troubleshooting and configuration options
